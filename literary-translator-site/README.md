# LiteraFlow

Отдельный новый сайт для литературного перевода страниц по ссылке.

## Что уже реализовано

- самостоятельный адаптивный сайт в папке `literary-translator-site/`
- mobile-first интерфейс с табами на мобильных и двумя колонками на десктопе
- ввод URL, выбор языка перевода, поле модели и расширенные настройки
- прогресс-бар, история переводов, экспорт результата в HTML
- извлечение основного текста из `article/main` с сохранением абзацев и простого форматирования
- контекстный перевод чанками через OpenAI Responses API
- память персонажей по именам и местоимениям
- кнопка литературного улучшения уже переведенного текста

## Как открыть

Откройте файл `literary-translator-site/index.html` в браузере.

## Важно

- прямой `fetch` внешних страниц из браузера часто упирается в CORS
- в этом случае сайт автоматически показывает демо-текст
- для production-версии стоит добавить серверный прокси, который будет забирать HTML страницы на бэкенде
- API key хранится только локально в браузере через `localStorage`
